# eMall 项目

一个**面向多角色的电商管理与购物平台**，支持商品、订单、用户等**全流程管理**。

---

## 1. 项目背景

随着线上购物需求的持续增长，传统电商系统已难以满足**多角色（管理员、商家、用户）协同管理**的需求。

本项目主要解决以下问题：

- 商品管理、订单处理、用户评价等电商核心流程的**数字化与自动化**
- 多角色操作下的**权限划分与协同效率**

**目标用户群体：**

- 中小型电商企业  
- 独立商家  
- 普通消费者  

旨在全面提升**管理效率**与**购物体验**。

---

## 2. 项目目标

- 实现商品、订单、用户、评价等业务流程的**可视化管理**
- 支持**多角色权限分离**，保障数据安全与操作便捷

**成功标准：**

- 系统稳定运行  
- 核心功能覆盖电商业务主流程  
- 用户操作流畅  
- 数据准确可靠

- Maven  
- IntelliJ IDEA  
- VS Code  
- Git  

---

## 4. 系统设计 / 架构

### 架构模式
- **前后端分离架构**
- 前端通过 **HTTP 请求** 与后端 **API** 通信

### 关键模块
- **用户管理**：注册、登录、权限控制  
- **商品管理**：增删改查、分类、收藏、浏览历史  
- **订单管理**：下单、购物车、订单评价  
- **管理后台**：广告管理、轮播图、统计报表
